About Me

A computer engineer with a passion for building technology that matters, bridging cutting-edge research with real-world impact.

Luca Di Grazia holds a Great Minds Fellowship at the University of St. Gallen in Switzerland, focusing on the security of AI-generated code, hosted by Prof. Dr. Guido Salvaneschi. Previously, he was a Postdoctoral Researcher at USI Lugano, working on GenAI for software testing and verification under Prof. Dr. Mauro Pezzè.

He earned his Ph.D. in Computer Science (summa cum laude) from the University of Stuttgart, supervised by Prof. Dr. Michael Pradel. His dissertation explored innovative approaches to software maintenance through search and predictions.

During a Research Internship at Uber in Amsterdam, he developed a GenAI-based bug-fixing approach that won an internal competition against 103 teams, with results presented directly to the Uber CEO.

Selected Publications

FSE 2026

AgentBound: Securing Execution Boundaries of AI Agents

AgentBound is the first access control framework for MCP servers with a policy enforcement engine that contains malicious behavior without requiring MCP server modifications.

ICSE 2026

E-Test: E'er-Improving Test Suites

E-Test leverages LLMs to identify untested execution scenarios and augments test suites accordingly, achieving an F1-score of 0.54 vs. 0.39 for vanilla LLMs.

ASE 2025

Do LLMs Generate Useful Concrete Test Oracles?

An empirical study using 13,866 unbiased test oracles from 135 Java projects to evaluate LLM effectiveness in generating test oracles.

TOSEM 2025

From Today's Code to Tomorrow's Symphony

Envisioning HyperAssistant, an AI tool for 2030 developers. Featured in The Register and Swiss IT Magazine.

ICSE 2024

PyTy: Repairing Static Type Errors in Python

An automated technique that resolves 85.4% of Python type errors using cross-lingual transfer learning, with high acceptance on GitHub PRs.

ECOOP 2024

The Fault in Our Stars: Reproducible Large-scale Code Analysis

A standardized methodology for sampling and studying repositories, emphasizing reproducibility and avoiding unreliable metrics.

ACM CSUR 2023

Code Search: A Survey of Techniques for Finding Code

A comprehensive overview of 30 years of research on code search: query processing, indexing, retrieval, and ranking techniques.

ESEC/FSE 2022 · Distinguished Paper Award

The Evolution of Type Annotations in Python

Analyzing 1.4M type annotation changes from 9,655 Python repositories. ACM SIGSOFT Distinguished Paper Award winner.

IEEE TSE 2022

DiffSearch: A Scalable Search Engine for Code Changes

A search engine retrieving relevant examples from millions of code changes within seconds. 2nd prize at ICSE SRC 2022.
